# AdventureWorkSales-Dashboard
Interactive Power BI dashboard analyzing sales performance using AdventureWorkSales dataset with insights on revenue, products, and regional trends.

#  Power BI Sales Dashboard
##  Project Overview
This project showcases an interactive **Power BI dashboard** built using the **AdventureWorks dataset**.
It provides meaningful insights into sales performance, product trends, and regional analysis.


##  Objectives

* Analyze overall sales performance
* Identify top-performing products
* Understand regional sales distribution
* Track trends over time


##  Files Included

* `AdventureWorksSales.pbix` → Main dashboard file
* `screenshot.png` → Dashboard preview


## Key Insights

*  Revenue trends over time
*  Region-wise sales analysis
*  Product category performance
*  Top & bottom performing products


##  Tools & Technologies

* Power BI Desktop
* Data Visualization
* DAX (Data Analysis Expressions)


##  How to Use

1. Download the `.pbix` file
2. Open it using **Power BI Desktop**
3. Explore interactive visuals


##  Learnings

* Data cleaning and transformation
* Creating interactive dashboards
* Using filters and slicers
* Writing DAX measures
